Basic Flask app

1. Flask is Microframework which means it’s SuperLightweight.
2. Doesn’t come prepackaged with anything at all.
3. You can get a simple web page running with only 6 lines of code.
4. The design philosophy is just a kind of Import thing as you need them.
5. So there is no database, no Form-validation.
6. Flask is very clean & minimalistic.
7. Unlike Django where everything bunched together which has database etc.
