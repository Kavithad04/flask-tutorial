Defining Database objects in Flask

The Flask-SQLAlchemy package makes use of SQLAlchemy to set up and inform the database structure.Database models consist of Columns, which are data types unique to sqlalchemy. An object could contain multiple Columns based on the business use case.SQLAlchemy will perform the CRUD operations on this object as an ORM.
