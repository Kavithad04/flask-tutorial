Deploying Flask App to Elastic Beanstalk

In this project we are going to Deploy the Flask Application in Elastic beanstalk environment.
The following process will help to complete the process

  • Create a Docker File
  • Eb CLI set up
  • Create an Application in AWS console
  • Create an Environment for the Application
  • Deploy an Application

    • Docker

  • Docker File - Build and run you image.



  • The below command will generate requirements.txt files where you can keep the required
    dependency for the project

    • “pip freeze > requirements.txt”




  • Verify Docker container is running successfully locally


    • AWS Elastic beanstalk CLI Set up


    Steps to Install EB CLI:

    1. Download Github if not available in your machine
    2. Clone this Repository https://github.com/aws/aws-elastic-beanstalk-cli-setup.git
    3. After installation go to the path “$cd aws-elastic-beanstalk-cli"
    4. Then, $cd script
    5. execute the ./bundled_intsaller



    Incase if you face issue regarding installation related to loading and linking OpenSSL and zlib.
    The following command installs the necessary packages

    brew install zlib openssl readline
    CFLAGS="-I$(brew --prefix openssl)/include -I$(brew --prefix readline)/include -
    I$(xcrun --show-sdk-path)/usr/include" LDFLAGS="-L$(brew --prefix openssl)/lib -
    L$(brew --prefix readline)/lib -L$(brew --prefix zlib)/lib"





    Create an Application on AWS console


    Before proceeding, Verify whether your AWS is configured by giving the below command on CLI
  • $aws configure



  • Login to AWS console and Navigate to services Elastic beanstalk - Create New Application



    • Steps to Configure the EB CLI



    To initialize the elastic beanstalk command in your application directory, follow the below steps

    $eb init - This command will give you series of questions:
    1. choose a region
    2. Select an application use or enter new application name
    3. It appears you are asking docker. is this correct> Y






    Create a new environment and deploy an application to it

    $eb create - This command will give you series of questions:
    1.Enter Environment name
    2.DNS CNAME prefix
    3.Select a load balancer




    The above process will deploy the code in Elastic beanstalk
    URL:http://flaskapplication.eba-kibpwfft.us-west-2.elasticbeanstalk.com/
