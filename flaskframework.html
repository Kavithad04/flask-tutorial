What is Flask Framework?

  • Flask is a Microweb Framework of Python. It is regarded as a highly flexible framework for developing web applications.

  • It is designed to be lightweight yet provides tools to extend.It also provides a lot of features and a huge number of extensions where users can change the Micro Framework into a full-featured Web application tool.

  • Some of the common extensions used in the framework are Flask - SQLAlchemy, Alembic, JINJA, and WSGI(Web server gateway Interface).Flask lets the developer take control and gives a variety of choices in terms of modules and add-ons when developing the application.

  • Though Flask is easy to start, the user can scale up the complexity according to the application. Hence it is a great option for all kinds of web application projects.
